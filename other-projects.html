Real Estate Data Evolution – Zipco

Streamlining property records with a PostgreSQL-powered ETL pipeline

In the ever-demanding real estate sector, timely and accurate property data is gold. For Zipco Real Estate, scattered data sources and inefficient workflows were hampering operational efficiency and slowing down decisions. So, I designed and deployed a robust ETL pipeline using PostgreSQL, Python, and AWS to bring order to chaos.

From fetching property records via API to transforming and cleaning data, then securely loading it into a PostgresSQL database, the pipeline ensured consistency, reliability, and scalability. By automating the entire process, Zipco could now access clean, queryable data in real-time—an essential edge in a competitive market.

  • Business Focus: Automated property data processing to reduce delays and increase decision accuracy
  • Key Learning Points: ETL architecture, API integration, data cleaning, database optimization, automation, and monitoring
  • Tech Stack: Python, SQL, PostgreSQL, AWS S3
  • Scope: API data ingestion, cleaning & transformation, optimized DB loading, pipeline automation with logging

Cohort Analysis for E-Shop Pro & Streamlix

Understanding retention patterns to reduce churn and improve engagement

In a bid to unravel the mystery behind E-Shop Pro’s declining customer retention, I turned to cohort analysis—a powerful technique for tracking user behavior over time. By segmenting customers based on their first purchase dates and analyzing how long they stayed active, I uncovered crucial retention insights that the marketing team had missed. Patterns of early churn were clear, and interventions were identified to enhance loyalty efforts.

The project also extended to Streamlix, where cohort insights guided improvements in content delivery. The analysis helped pinpoint how viewer engagement changed over their user lifecycle—information that fed directly into better curation and experience design.

  • Key Business Outcomes: Identified key churn points, optimized campaign timing, and refined retention strategies.
  • Tech Stack: Python (NumPy, pandas, seaborn, matplotlib, scikit-learn)
  • Scope: Exploratory Data Analysis, Feature Engineering, Predictive Modeling, Model Evaluation
  • Data Points: Customer ID, Invoice Date, Quantity, Country, Product Info, Pricing, and Purchase Frequency
